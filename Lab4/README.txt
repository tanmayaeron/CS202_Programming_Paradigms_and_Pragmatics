Assumption - only one letter can be guessed at a time.

Run program by typing perl Hangman.pl in command line.
You get six chances to guess the word and lose one chance on wrong guess.
Guess a letter. Program asks to enter letter until you make a legitimate guess.
After you make a guess, program tells whether it was a good guess or not.
Game gets over when user either guesses word correctly or run out of chance.
After game is over, program ask for another game.
